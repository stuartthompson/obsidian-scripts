# Obsidian Scripts

This repository contains scripts designed to generate files for use in [Obsidian](https://obsidian.md/), a powerful knowledge management and note-taking application. These scripts automate the creation of structured notes, making it easier to organize and manage your data within Obsidian.

## Features

- **Premier League Fixture Converter**: A script to process and convert Premier League fixtures into a format suitable for Obsidian notes.
- **Polar Beat Converter**: A script to convert Polar Beat data into Obsidian-compatible notes.

## Usage

1. Clone this repository to your local machine.
2. Run the desired script using Node.js.
3. Follow the prompts or customize the scripts to suit your specific needs.
